# Japan xAPI Profiles (Draft)

このリポジトリは、日本の教育DXにおける学習ログ（xAPI）の標準化を目指す「Japan xAPI Profiles」のドラフト版を管理しています。

## 概要

多様な教育プラットフォームやツール間で学習データを相互運用可能にするため、日本国内での利用を想定したxAPIプロファイルの定義を行っています。
本プロファイルは、各ドメイン（領域）ごとに分割して管理されています。

## リポジトリ構成

各ディレクトリは、特定のドメインごとのプロファイル定義を含んでいます。

- **core**: 共通的に利用される語彙や定義を含むコアプロファイル
- **cbt**: Computer Based Testing (CBT) に関連するプロファイル
- **ebook**: デジタル教科書・教材（eBook）に関連するプロファイル
- **lms**: 学習管理システム (LMS) に関連するプロファイル
- **group**: グループ学習や協働学習に関連するプロファイル

## ステータス

現在策定中のドラフト版です。内容は変更される可能性があります。
